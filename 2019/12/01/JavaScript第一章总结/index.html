JavaScript第一章总结

JavaScript第一章 概述
JavaScript特点：
1.向HTMl页面添加交互行为（如注册验证）
2.属于弱类型语言（任何类型变量都用var声明）
3.编写客户端脚本（写在页面上，对网站页面起作用）
4.解释型语言，边执行边解释（不需要编译）

JavaScript诞生于1995年，由网景公司研发。

JavaScript由三部分组成：ECMAScript（行业标准、规范）、BOM（浏览器对象）、DOM（文档对象）

ES6=ECMAScript2015
ES7、ES8都是升级的高版本。

JavaScript引入到网页中的三种方式：
1、
2、标签
3、在HTML属性中直接引入（通过事件触发js特效）
<input type=”button” value=”弹出消息框” onclick=”alert(‘hello world’!)”>

基本语法：
声明变量都用var关键字。
标识符命名规则:
1.不能是关键字和保留字（现在不是关键字，随着版本更新，可能会被当做关键字的字符）
2.只能包括数字、字母、_、$
3.不能以数字开头

js中三种对话框：
1.alert();只用来提示，没有返回值
2.confirm();确定或取消，返回true或false
3.prompt(提示信息,默认值);接收用户输入的信息，返回值是用户输入的信息，若没有输入，返回默认值。如果点击取消或×，返回null。
