java第五章总结

循环结构：

1.
定义循环变量
while(条件) 如果条件为true，才执行循环体
{
循环体;
改变循环变量;
}

2.do
{循环体}
while(条件);

两个循环的区别：
while是先进行条件判断，再循环。
do-while先循环一次，在进行条件判断。

程序调试：
1、设置断点 （双击代码行号左边相应需要暂停的地方）
2、进入调试模式 （点击绿色的小虫子Debug）
3、F5单步跳入、F6（常用） 单步跳过

equals();方法比较两个字符串的值是否相同。
