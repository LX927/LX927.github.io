JavaScript第6章复习

JavaScript函数声明方式：
var 变量名 = function(参数){
方法体;
return 返回值;
}
必须先声明定义函数，再调用

function 函数名(参数){
方法体;
return 返回值;
}
可以先调用，后声明定义

如果方法的参数个数不确定的话，可以用arguments对象来获取传入的所有实参。
它相当于一个数组，通过length获得长度，通过循环来获得传入的每个实参。

局部变量：函数function中定义的变量
全局变量：函数function外定义的变量（function中声明定义时不加var）
