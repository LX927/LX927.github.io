JavaScript第七章重点总结

第七章 对象
1.对象是单个实物（一个类）的抽象（抽离其特征，形成对象）
2.对象是一个容器，封装了属性和方法。（对象就是一个载体，由属性和方法组成）

创建对象：
1.通过字面量创建
var 对象名 = {
属性名:属性值,
方法名:function(){
方法体;
}
}

2.通过内置的Object对象创建
var 对象名 = new Object();
对象.属性=属性值;
对象.方法名=function(){
方法体;
}

构造函数：就是生成对象的模板，减少相同代码的代码量。就相当于一个类。
function Student(name,studentNum,sex){
//this指代当前生成的对象
//this.name指代当前对象的name属性
//=name传进来的属性值
this.name=name;
this.studentNum=studentNum;
this.sex=sex;
this.hello=function(){
alert(this.name+“说你好”);
}
}
var a = new Student(“小明”,20180306xxxx,”男”);

访问（调用）对象属性和方法：
console.log-F12在浏览器控制台输出
document.write–在HTML页面上输出
alert(对象名.属性名)-通过弹出框输出
对象名[“属性名”]

对象名.方法名();

var xiaohong = new Object();
xiaohong.sex=”女”;
xiaohong.age=18;
xiaohong.name=”小红”;
xiaohong.eat=function(){
alert(“一日三餐”);
}

属性的简洁表示法：
var sex=”女”;
var age=18;
var name=”小红”;
var xiaohong={sex,age,name,eat(){alert(“一日三餐”);
}}

原型与原型链：
原型：JavaScript的每个对象都继承另一个对象，后者父对象称为“原型对象”。除了null之外，都有原型对象。
类名.prototype

继承：如果自己没有某些属性和方法，就调用父对象的属性和方法。
多态：如果自己和父对象都有某些属性和方法，优先调用自己的属性和方法。（重载）

遍历对象：
for in结构来遍历对象：
x代表此对象所有的属性名和方法名。
对象名[x]代表此对象所有的属性值和方法体。
for(var x in 对象名){
console.log(x+”=”+对象名[x]);
}
