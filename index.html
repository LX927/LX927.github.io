JavaScript第八章重点总结

1.定义数组：
var 数组名 = new Array();
数组名[0]=元素1;
.
.
.

var 数组名 = new Array(元素列表);

var 数组名 = [元素列表];

2.数组调用通过下标，下标都从0开始，通过length获取数组长度。

indexOf(),搜索指定元素首次出现在数组的位置。
concat(),拼接数组，返回拼接成功的新数组。
join(),把数组所有元素连接在一起，形成一个字符串。
pop(),删除数组的最后一个元素，返回删除的那个元素。
shift();把数组的第一个开始删除，返回值是第一个元素的值。
push(),在数组的末尾添加一个或多个元素，返回新数组的长度
unshift();在数组的开头添加一个或多个元素，返回值是新数组的长度

slice(),在原数组中截取一部分，返回截取后的新数组。左闭右开
splice(),不仅能删除，还能添加元素。
第一个参数代表开始操作的位置，
第二参数为0代表不删除，为正数代表删除元素的个数。
从第三个参数开始，都代表要添加的新元素。
sort();数组元素自动排序，可以规定是升序也可以规定降序。
arr.sort(function(a,b){return a-b;});—升序
arr.sort(function(a,b){return b-a;});—降序

reverse();将原数组的元素位置反向。

JavaScript第三章总结

JavaScript第三章 运算符和表达式
一元运算符：++、–（本质上就是加1或减1）
前置和后置：前置先进行自加或自减，再赋值
后置先赋值，再进行自加或自减

算数运算符：
+:正常加法运算，拼接作用
-
*
/
%
共同特点：如果不是Number类型，进行运算的时候会自动进行类型转换。
var result = 100/””;——–结果是Infinity
var result=100/null;——-结果也是Infinity

var result=100%””;——-结果是NaN
var result=100%null;——-结果也是NaN

关系运算符：
 >= < <= == != === ！==
 如果是不同数据类型之间用关系运算符比较，会自动转换成统一数据类型，再比较（Java中不同数据类型之间无法比较）
 两个字符串进行比较的话，取首字符的ASCII码表中对应的值进行比较。如果首字符相同的话，对下一位进行比较。
 ==和!=比较的时候如果数据类型不同的话，会先进行类型转换，然后再比较
 ===和!==比较的时候不会进行数据类型转换。

逻辑运算符：
逻辑或、逻辑与的结果不一定是布尔值：
||:第一个操作数转换成true，返回第1个操作数。
第一个操作数转换成false，返回第2个操作数。
&&:第一个操作数转换成true，返回第2个操作数。
第一个操作数转换成false，返回第1个操作数。

三目运算符：
判断条件表达式?表达式1:表达式2;
如果条件为true，执行表达式1，否则执行表达式2.
